GIT COMMANDS:

Clonar Repositorio: 
git clone {url repositorio}

Ver Ramas:
git branch

Crear Rama:
git branch {nombre rama}

Moverte a Rama
git checkout {nombre rama}

Eliminar Rama
 -> Opción -d: para eliminar de forma segura (si la rama ya ha sido fusionada)
	git branch -d {nombre de la rama}
 
 -> Opción -D: eliminar rama forzadamente (sin importar si la rama se ha fusionado)
	git branch -D {nombre de la rama}

Ver Cambios 
git status

Agregar Cambios
git add . (. son todos los archivos modificados)

Descartar Cambios (No hace falta)
git checkout . (en vez del punto se podrian poner los archivos a descartar)

Commit/Confirmación de los Cambios:
git commit -m "{Mensaje explicativo del cambio (corto)}"

Ver Commits Pendientes
git log

Ver diferencia entre commits 
git diff {id1} {id2}
git push


PipeLine Branch:

Creamos una nueva rama:
	-> git checkout main
	-> git pull origin main #Obtenemos ultima version
	-> git checkout -b new_Feature #-b nos lleva directamente a esa rama

Desarrollamos código en esa rama

Realizamos cambios: 
	-> git add .
	-> git commit -m "Implementar nueva cosa"
Subimos cambios a la rama de github
	-> git push origin new_Feature

En este punto podemos hacer dos cosas, o creamos un pull request desde la web, o hacemos un merge.

Pull Request:
	EN la web del repo ir a Compare & pull request junto a la nueva rama
	Añadir titulo descriptivo y descripccion de los cambios
	Los compis revisan los cambios y luego se fusiona, dandole a Merge pull request

La otra opción es hacer una fusión sin pull request: 
	1. Ir a la rama principal y actualizarla
		git checkout main
		git pull origin main
	2. Fusionar rama con main
		git merge new_Feature # new_Feature es el nombre de la rama

	Ahora se resuelven conflictos si los ahi
	3. Guardar cambios
		git add . 
		git commit -m "We did it man"		
	4. Empujar cambios fusionados a github

Ahora si queremos eliminar la rama:
	Localmente: 
		git branch -d nombre-de-la-rama    o -D si no se ha fusionado completamente

	Repositorio Remosto:
		git push origin --delete nombre-de-la-rama
